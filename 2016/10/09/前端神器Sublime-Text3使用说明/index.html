前端神器Sublime Text3使用说明

关于软件

Sublime官网
这里安装的是Sublime Text3

常用插件安装

通过直接将压缩包解压到Packages目录下：Menu->Preferences->pakages
可以使用Pacakge Control组件进行安装，可以直接在线安装插件

  1. 按Ctrl + ` 调出Console

  2. 执行下面的命令

  3. 使用PackageControl 进行插件的安装
    Ctrl + Shift + P 调出命令面板
    输入install package 调出可以安装的插件，进行安装

使用说明

  1. 修改主题
    可以通过安装插件的方式进行安装，这里有主题的列表
    选择一个自己喜欢的主题，以插件的方式进行安装

  2. 多行游标
    在Sublime Text3中可以使用多行游标。使用多行游标重点是进行游标的选择。

    Ctrl + D 进行游标选择
    Ctrl + K, Ctrl + D 可以跳过一个选择
    Alt + F3 可以多选多个游标
    首先选择多行 Ctrl + Shift + L 在多行形成游标

  3. 命令模式

    Ctrl + Shift + P 打开命令模式，支撑模糊查找。

  4. 跳转命令

    Ctrl + P 可以跳转到任何地方
    可以在打开的输入框中输入文件名
    可以在css文件中@+选择器名
    可以在html文件中#+标签名

  5. Emmet插件
    这个插件是前端书写的利器，可以使用简单的输入和Tab完成基本的页面书写
  6. 编辑操作

    Ctrl + [ 代码右移动
    Ctrl + ] 代码左移动
    Ctrl + Shift + V 格式化粘贴

  7. 快捷输入
    JavaScript & NodeJS Snippets 使用这个包可以增加输入速度

8.添加注释

DocBlocker 插件：在需要添加注释的地方输入 /** +tab

9.颜色选择器

ColorPicker 可以进行颜色的选择：Ctrl + Shift + C

快捷键大全

快捷键 说明
Ctrl+Shift+V 粘贴并自动缩进(其它兄弟写的，实测win系统自动缩进无效)
Ctrl+M 光标跳至对应的括号
Alt+. 闭合当前标签
Ctrl+Shift+A 选择光标位置父标签对儿
Ctrl+Shift+[ 折叠代码
Ctrl+Shift+] 展开代码
Ctrl+KT 折叠属性
Ctrl+K0 展开所有
Ctrl+U 软撤销
Ctrl+T 词互换
Ctrl+Enter 插入行后
Ctrl+Shift Enter 插入行前
Ctrl+K Backspace 从光标处删除至行首
Ctrl+Shift+UP 与上行互换
Ctrl+Shift+DOWN 与下行互换
Shift+Tab 去除缩进
Tab 缩进
F9 行排序(按a-z)
Alt+Shift +１　 恢复默认分屏
Alt+Shift +2　 2个分屏　
Ctrl + ` 显示隐藏左侧浏览栏
